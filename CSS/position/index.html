RULE #1 - DO NOT ABUSE POSITION

Reminder: All values of the position property are relative. It just depends on what they are relative to.

Static: The default value for every element's position property. Top, right, bottom, left will have no effect.

Relative: The element is positioned relative to its normal position.

Absolute: The element is position relative to its closest positioned ancestor.

When you are absolutely positioning an element, you should also relatively position its parent.

Fixed: The element is positioned relative to the viewport.

Please accept my cookies!

Sticky: The element will toggle between fixed and relative depending on the scroll position.
