Document Object Model (DOM)

The DOM is the browser window represented as an object.

A JavaScript object has properties and methods, just like object IRL have characteristics and things it can do.

Car IRL Object

Car characteristics (JS properties):

  • Color
  • Make
  • Number of wheels

Things a car can do (JS methods, a method is just a function that belongs to an object)

  • Drive
  • Beep

Onclick is a special kind of attribute - it's an event listener attribute
